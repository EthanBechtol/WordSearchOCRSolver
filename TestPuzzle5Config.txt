If you encounter a "TesseractNotFoundError", add the following (and change the name to your user name):
-tesseract C:\Users\YOUR_NAME_HERE\AppData\Local\Tesseract-OCR\tesseract.exe

This is the default installation location for most users, however if you installed it elsewhere, change the path.
